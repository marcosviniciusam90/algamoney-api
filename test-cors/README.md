## Executando aplicação para testar CORS 

1. Baixe o Python em: https://www.python.org/
2. Na instalação marque a opção **Add Python to environment variables**.
3. Após a instalação, abra o prompt e verifique se o Python foi instalado, pelo comando **python -V**.
4. Agora vamos subir o servidor, navegue até o diretório que contenha a página HTML e execute o comando **python -m http.server**. 
Por padrão, o servidor é executado na porta 8000.
